A robust, enterprise-grade web application developed during my **Web Developer Internship** at **Zeel Network Solutions**. This system automates corporate administrative workflows, manages employee lifecycles, and provides real-time data insights.

---

## 🚀 Key Features
- **Secure Authentication:** Robust login system with session management and parameterized queries to prevent SQL Injection.
- **Employee Lifecycle Management:** Full CRUD (Create, Read, Update, Delete) operations for managing employee, department, and role records.
- **Advanced Reporting:** Integrated **SQL Joins** to display relational data across multiple entities in a unified view.
- **Responsive UI:** Modern interface built using **ASP.NET Master Pages** and **Bootstrap 5** with a clean corporate aesthetic.
- **Data Persistence:** Implemented **ADO.NET Disconnected Architecture (DataSet)** for efficient and reliable database interactions.

## 🛠 Tech Stack
* **Frontend:** ASP.NET Web Forms, HTML5, CSS3, Bootstrap 5
* **Backend:** C# (C-Sharp)
* **Database:** Microsoft SQL Server
* **State Management:** Session, ViewState
* **Architecture:** N-Tier Architecture (Presentation & Data Layers)

## 📂 Project Structure
* `MasterPages/`: Contains the global layout and navigation.
* `Pages/`: Individual modules for Employee Master, Dept Master, and Search.
* `App_Data/`: SQL scripts for database restoration.
* `Web.config`: System configuration and connection string settings.
